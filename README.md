# Laravel 6 App setup

## with Auth, Mix, Vue, VueRouter, Vuex, Sass, Bootsrap, BootstrapVue, PhpCS and IDE helper

```bash
composer install
yarn install or npm install 
php artisan key:generate
# after setting database in .env
php artisan migrate
php artisan ide-helper:generate 
# Done
```
